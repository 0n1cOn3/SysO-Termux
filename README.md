# SysO-Termux
#
# ¿Qué es SysO-Termux?
#
SysO-Termux, con sus siglas, "Termux Operating System", es un instalador de paquetes, herramientas y frameworks, específicamente para "Termux", con el fin de simular la terminal del sistema operativo "Kali Linux" y así poseer la facilidad de usar algunas de estas herramientas adaptadas para android, además, incluye un banner, una shell diferente conocida como "fish", un teclado adicional con teclas útiles, y un login con "user" y "password" a elección del usuario.

# HERRAMIENTAS
#
[01]💲shellphish

Es una herramienta que se utiliza para robar las credenciales de un usuario a través de phishing utilizando 18 plantillas preconfiguradas.

[02]💲saycheese

Es una herramienta que captura fotografías de la cámara frontal del dispositivo víctima a través de un enlace.

[03]💲sqlmap

Es una herramienta desarrollada en python para realizar inyección de código sql automáticamente. Su objetivo es detectar y aprovechar las vulnerabilidades de inyección SQL en aplicaciones web.

[04]💲FotoSploit

Es una herramienta que permite subir una imágen a un enlace phishing, por lo tanto, las probabilidades de que el objetivo ingrese al enlace aumentan.

[05]💲hammer

Es una herramienta de ataques "Denegación de Servicio" que permite acabar con el servidor web de otra máquina con un mínimo de efectos secundarios en ancho de banda, servicios no relacionados y los puertos.

[06]💲SocialSploit

Es una herramienta que se utiliza para robar las credenciales de un usuario a través de phishing utilizando 21 plantillas preconfiguradas.

[07]💲seeker

Es una herramienta que captura la ubicación exacta de un dispositivo, a través de un enlace generado por serveo.

[08]💲MagmaOsint

Es una herramienta que sirve para buscar y recopilar información de un objetivo, con solo su nombre o nick.

[09]💲routersploit

Es un framework escrito en python, que nos permitirá comprobar la seguridad de nuestros routers, puntos de acceso, NAS y otros dispositivos embebidos.

[10]💲phonia

Es una de las herramientas más avanzadas para escanear números de teléfono, utilizando solo recursos gratuitos.

[11]💲kickthemout

Es una herramienta que nos permite expulsar dispositivos conectados a nuestra red y de ésta manera, negar la posibilidad de consumir ancho de banda a los intrusos.

[12]💲fbi

Es un herramienta de información gathering precisa de una cuenta de Facebook. Toda la información confidencial se puede recopilar fácilmente a pesar de que el objetivo/víctima convierta toda su privacidad a (solo para mí), información confidencial sobre residencia, fecha de nacimiento, ocupación, número de teléfono y dirección de correo electrónico.

[13]💲TBomb

Es una herramienta que realiza spam de SMS a un número telefónico anónimamente sin necesidad de poseer saldo o crédito.

[14]💲websploit

Es una herramienta que sirve para escanear y analizar sistemas remotos para encontrar varios tipos de vulnerabilidades.

[15]💲weeman

Es una herramienta que copia el código html de una página web a tu elección, y crea un enlace phishing con dichos datos.

[16]💲cupp

Es una herramienta que nos permite generar diccionarios con información de personas, generalmente esta herramienta se usa para los ataques de fuerza bruta.

[17]💲PortmapSploit

[18]💲Facebook_brute

Es una herramienta que realiza ataques de fuerza bruta a cuentas de Facebook con solo escribir el ID de tu objetivo.

[19]💲scam

Es una herramienta de phishing moderna con plantillas actualizadas, en especial, para la red social Facebook.

[20]💲quack

Es un conjunto de herramientas para proporcionar ataques de denegación de servicio.  Quack Toolkit incluye herramienta de ataque SMS, herramienta de ataque HTTP y muchas otras herramientas de ataque.

[21]💲formphish

Es una herramienta que copia el código html de una página web a tu elección y de ésta forma, brinda un enlace phishing automáticamente utilizando el código de la página web.

[22]💲jager

Es una herramienta que captura la ubicación exacta de un dispositivo objetivo a través de un enlace con una plantilla de ingenieria social.

[23]💲vigo

Es una herramienta que captura fotografías de la cámara frontal del dispositivo objetivo a través de un enlace con una plantilla de ingenieria social.

[26]💲DoxWeb

Es una herramienta que utiliza diversos sitios web para recopilar información de un usuario, es decir, es una herramienta de Doxing.

[27]💲Geonumb

Es una herramienta avanzada para escanear, analizar y recopilar información de un número telefónico utilizando recursos gratuitos.

[28]💲HiddenEye

Es una herramienta de hacking todo en uno que incluye Keylogger, Ingeniería social y Phishing para más de 30 redes sociales, entre las que destaca Facebook, Twitter, Instagram.

[29]💲AresBomb

Es una herramienta que realiza spam de SMS a un número telefónico, sin necesidad de poseer saldo o crédito.

Es una herramienta que utiliza w3m para abrir la página web de correos temporales mejor conocida como "TempMail" en la terminal termux, y de ésta forma, obtener un correo electrónico temporal o desechable.

[31]💲ShortUrl

[32]💲hacklock

[33]💲Falsify

Es una herramienta de phishing con 8 plantillas de redes sociales, y la opción que permite enviar un mensaje phishing de Facebook por correo electrónico (Gmail) con un mensaje falso de "Alerta de Eliminación de la Cuenta de Facebook".

[34]💲ngrok

Es una herramienta que permite acceder nuestro servidor local a cualquier persona en internet con la que compartamos una url generada dinamicamente.

[35]💲metasploit

Es un proyecto de código abierto para la seguridad informática, que proporciona información acerca de vulnerabilidades de seguridad y ayuda en tests de penetración "Pentesting" y el desarrollo de firmas para sistemas de detección de intrusos.

[36]💲bettercap

Es una suite de herramientas basada en la original Ettercap que nos va a permitir analizar el tráfico de nuestra red, controlarlo y poder auditar la seguridad de una red y de los datos que viajan por ella.

[37]💲gophish

Es una herramienta diseñada para lanzar campañas de phishing simuladas, monitorizadas y analizadas para ver los resultados.

The Broswer Exploitation Framework, a diferencia de otras herramientas de pentesting web, se centra en el contexto de los clientes y utilizando un hook en Javascript, permite crear una red de bots que pueden ser controlados desde un panel de control central.

[39]💲shodan

Es un motor de búsqueda en el que, a diferencia de Google y otros buscadores, no podemos buscar, por ejemplo, una imagen o un texto. Este motor de búsqueda está enfocado únicamente a buscar sistemas y servicios conectados a internet.

[40]💲hydra

[41]💲RED_HAWK

[42]💲nmap

[43]💲tor
